# 역병예보 (EpiWeather)

> 코로나 때 잃어버린 5~8주를 되찾는 AI 조기경보 통합 관제센터.

## 무엇인가요?

해외 발원지부터 한국 국내 확산, 그리고 방어 대응까지 — 감염병 위협의 전 과정을 한 화면에서 추론·예측·대응하는 통합 시스템입니다. 코로나의 비극을 반복하지 않기 위해 만들었습니다.

## 5분 안에 시작하기

### 1. 메인 화면 보기

```bash
# 브라우저로 열기
open prototypes/01-unified-command.html
```

좌측 사이드바로 **7단계**(발원지 추적 → 민간 신호 → 글로벌 유입 → 국내 확산 → 도착 예측 → 방어 대응 → AI 추론)를 이동하면서 시나리오를 탐색하세요. 상단 발원지·위협·대응을 바꾸면 7단계가 동시에 갱신됩니다.

### 2. Python 엔진 돌려보기

```bash
cd engine
pip install pandas numpy scipy --break-system-packages

# 데이터 수집·정렬 + 교차상관 분석
python3 run_pipeline.py

# 후향 검증 (Farrington vs z vs EWMA)
python3 run_backtest.py

# 통합 실행
python3 run_full.py
```

합성 데이터로 즉시 작동하며, 환경변수로 API 키를 넣으면 실제 데이터로 전환됩니다.

```bash
export KDCA_API_KEY="공공데이터포털_키"
export NAVER_CLIENT_ID="네이버_개발자센터_키"
export NAVER_CLIENT_SECRET="네이버_시크릿"
```

### 3. 사업 맥락 읽기

- `docs/strategy.md` — 15장 사업 전략서 (시장·BM·데이터·취약계층·신종감시·대응체계)
- `docs/phase0-poc-plan.md` — 90일 PoC 실행계획
- `docs/architecture.md` — 시스템 아키텍처와 다음 단계

```
prototypes/   ← 7개 HTML 프로토타입 (01번이 메인)

docs/         ← 전략·계획·아키텍처 문서
scripts/      ← 검증 스크립트
CLAUDE.md     ← Claude Code용 컨텍스트 (개발 시 첫 읽기)
```

## 핵심 알고리즘

- **Farrington 이상탐지** — 공중보건 표준, 로그선형 추세 + 과분산 보정
- **피셔 결합법** — 약한 다채널 신호를 통계적으로 증폭 (스컹크웍스 원리)
- **SEIR + 시간가변 개입** — 방역 조치가 확산 곡선을 어떻게 꺾는지
- **메타개체군 확산** — 지역 간 이동량 위에서 도착 시점 예측
- **베이지안 시점 역추론** — 현재 관측에서 과거 발원 시점 추정

## 검증된 성능 (합성 데이터)

| 지표 | 값 |
|---|---|
| 선행 신호 융합 vs 공식 발표 | **+8~9일 선행** |
| Farrington vs 단순 z-score | 평균 **3~5일 일찍** 잡음 |
| 발원지 격자 식별률 (6채널 결합) | **90%** (단일 채널은 59%) |
| 강력 개입 시 사망 감소 | **99%+** |
