## 3. 단계별 마이그레이션 계획

### Phase 1 — API화 (2~4주)
**목표**: 프로토타입의 모든 계산을 FastAPI 엔드포인트로

- [ ] `engine/src/` 의 함수를 그대로 FastAPI 라우터로 노출
- [ ] Pydantic 스키마 정의 (시나리오·결과 타입)
- [ ] OpenAPI 자동 문서화
- [ ] Docker 컨테이너화
- [ ] 단위 테스트 (pytest) — 합성 데이터 기준값 정착

**산출물**: `localhost:8000/docs` 에서 모든 엔드포인트 확인 가능

### Phase 2 — 프로토타입 → React (4~8주)
**목표**: 01번 통합 관제센터를 production 코드로

- [ ] Next.js 14 App Router 셋업
- [ ] 시나리오 상태 관리 (Zustand)
- [ ] 7단계 페이지 컴포넌트 (route: `/stage/[id]`)
- [ ] SVG 시각화는 D3 또는 Recharts로 재작성
- [ ] API 호출 (React Query)
- [ ] 모바일 반응형
- [ ] Vercel 배포

**원칙**: 프로토타입의 모든 인터랙션·디자인을 유지. 다크 테마, 사이드바, 시나리오 공유 모두 그대로.

### Phase 3 — 실제 데이터 연결 (4~6주, Phase 1과 병행 가능)
**목표**: 합성 폴백을 진짜 데이터로 교체

- [ ] 질병청 감염병포털 ILI Open API 신청·연동
- [ ] 네이버 데이터랩 검색어 트렌드 API
- [ ] 하수 역학 데이터 — 지자체 1~2곳 파일럿 제휴
- [ ] 데이터 품질 모니터링 대시보드

**중요**: API 키 누출 방지. 환경변수·Secret Manager.

### Phase 4 — 시민 신호 수집 (6~10주)
**목표**: B2C 모바일 앱으로 시민 자가증상 수집

- [ ] React Native 또는 Flutter 모바일 앱
- [ ] 익명 등록 (전화번호 또는 이메일 없이도 동작)
- [ ] 차등 프라이버시 적용 (Apple/Google이 쓰는 방식)
- [ ] 데이터 활용 동의·투명 공개
- [ ] 보상: 사용자가 가장 정확한 동네 위험 정보를 가장 먼저 받기

### Phase 5 — 정부·의료기관 자동 통보 (2~4주, Phase 1 이후)
- [ ] 임계 초과 시 SMS·이메일·웹훅 자동 발송
- [ ] 질병청·지자체·병원 통보 라우팅
- [ ] 사전 보고서 PDF 자동 생성 (Claude API)

## 4. 기술 스택 추천

| 영역 | 1순위 | 대안 | 이유 |
|---|---|---|---|
| **Frontend** | Next.js 14 | Remix | App Router, 서버 컴포넌트, SEO |
| **상태관리** | Zustand | Jotai | 7단계 시나리오 공유에 적합 |
| **차트** | D3 + Recharts | Chart.js | SVG 직접 제어 필요 |
| **Backend** | FastAPI | Flask | Pydantic, async, 자동 OpenAPI |
| **DB** | PostgreSQL + TimescaleDB | InfluxDB | 시계열 + 관계형 둘 다 필요 |
| **잡 큐** | Redis + RQ | Celery | 가벼움 |
| **데이터 수집** | Airflow | Prefect | 표준, 대시보드 |
| **AI** | Claude Sonnet 4.6 API | GPT-4 | 한국어, 추론 품질 |
| **배포** | Vercel (FE) + Railway (BE) | AWS ECS | 빠른 MVP |
| **모니터링** | Sentry + Grafana | Datadog | 비용 |

## 5. 데이터 모델 (초안)

```python
# Signal — 모든 입력 신호의 표준화 표현
class Signal(BaseModel):
    source: Literal["search", "otc", "sewer", "animal", "clinical", "citizen", "official"]
    timestamp: datetime
    region_id: str               # 시·도 또는 시·군·구
    grid_id: Optional[str]       # 1km² 격자 (있을 경우)
    value: float                 # 정규화된 값
    confidence: float            # 0~1
    raw_metadata: dict           # 출처별 원본 메타

# Alert — 엔진이 생성하는 경보
class Alert(BaseModel):
    id: UUID
    timestamp: datetime
    level: Literal["관심", "주의", "경계", "심각"]
    origin_grid: Optional[GridCoord]
    origin_probability: float
    affected_regions: list[str]
    estimated_arrival_days: dict[str, int]
    threat_level: ThreatLevel
    recommended_actions: list[str]
    confidence: float

# Scenario — 시뮬레이션 입력
class Scenario(BaseModel):
    origin_city: str
    threat_id: Literal["flu", "novel", "severe"]
    intervention_levels: InterventionLevers
    civic_sources_enabled: list[str]
```
