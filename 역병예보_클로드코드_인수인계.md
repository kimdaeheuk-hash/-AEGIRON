## 경쟁 시스템과의 차이

```
WHO:       공식 신호만 봄 → 항상 늦음
BlueDot:   뉴스 + 항공편 봄 → 유전체 분석 없음
HealthMap: 뉴스 자동 수집 → 한국 특화 없음

역병예보:  6계층 신호 + 부정적 공간 감시 + 동물신호
           + 합성위협 탐지 (경쟁 없는 영역)
           → Global Anomaly Index 단일 점수
           → "설명 안 되는 것을 먼저 경보"
```

### 🔴 Phase 1 — 즉시 구현 (백엔드 핵심)

#### ① 6계층 신호 구조 + Global Anomaly Index

```
신호 계층:
  1층 공식신호   WHO/CDC/KDCA 공식 발표
  2층 비공식신호 현지 뉴스·지역 커뮤니티·보건 공지
  3층 행동신호   검색량·약품구매·병원방문·학교결석
  4층 환경신호   기후·모기지수·수질·대기질
  5층 동물신호   가금류폐사·야생동물 이상·수의학 보고
  6층 설명불가   원인불명 클러스터·부정적 공간

Global Anomaly Index (GAI) =
  공식신호   × 0.15
+ 비공식신호 × 0.20
+ 행동신호   × 0.25  ← 가장 중요 (공식 발표 전에 먼저 변함)
+ 환경신호   × 0.15
+ 동물신호   × 0.15
+ 설명불가   × 0.10
= 0~100점

70점↑ → 🟡 주의
80점↑ → 🟠 경보
90점↑ → 🔴 위험
```

#### ② Risk Score 개별 공식

```python
# 각 신호원별 이상도 (표준편차 기반)
검색트렌드_이상도 = (오늘값 - 30일평균) / 30일표준편차
뉴스빈도_이상도   = 오늘_언급수 / 30일_평균_언급수
동물질병_이상도   = 이번주_보고건수 / 4주평균
증상언급_이상도   = 동시급증_증상키워드수 / 전체_키워드수
환경_이상도       = (모기지수 + 기온이상 + 강수이상) / 3

# 개별 점수 → 정규화(0~100) → GAI 가중 합산
```

#### ⑨ 국가별 위험 지수 + 취약성 공식

```
국가 취약성 지수 =
  (1 - 의료인프라지수) × 0.30   ← WHO Global Health Observatory
  + 인구밀도지수       × 0.25   ← World Bank API
  + 공항연결성지수     × 0.25   ← OpenFlights (한국 유입 위험)
  + 국경이동량지수     × 0.20   ← UNWTO 관광 데이터

최종 국가위험도 = 원시위험도 × 국가취약성지수

예시:
  DRC     95점 🔴  (에볼라 PHEIC)
  우간다  88점 🔴  (에볼라 접경)
  사우디  72점 🟠  (MERS 집단감염)
  태국    65점 🟠  (뎅기열 급증)
  한국    38점 🟡  (해외유입 주의)
```
